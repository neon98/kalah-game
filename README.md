# kalah-game
Kalah game web application using Angular and Spring boot

# Prerequisites
- Install Java
- Install maven
- Install Node
- Install Angular-cli


# Project setup 

## Client
Navigate to `/client` folder
```bash
cd client
```
### Install
```bash
npm install
```

### Run

```bash
ng serve
```
Navigate to `http://localhost:4200/`

## Server
Navigate to `/server` folder
```bash
cd server
```
### Build
```bash
mvn clean install
```

### Run
```bash
mvn spring-boot:run
```
# Dependencies
- Java : 1.8
- Node : 17.7.2
- npm : 8.5.2
- Angular : 13.3
